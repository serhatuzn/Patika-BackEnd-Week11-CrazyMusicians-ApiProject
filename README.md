## üöÄ How to Run

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd CrazyMusicians
   ```

3. Restore dependencies and build the project:
   ```bash
   dotnet restore
   dotnet build
   ```

4. Run the project:
   ```bash
   dotnet run
   ```

5. The API will be available at `http://localhost:5000`.

## üìñ API Endpoints

- `GET /api/musicians`: Retrieve all musicians.
- `GET /api/musicians/{id}`: Retrieve a musician by ID.
- `POST /api/musicians`: Add a new musician.
- `PUT /api/musicians/{id}`: Update a musician by ID.
- `DELETE /api/musicians/{id}`: Delete a musician by ID.
- `PATCH /api/musicians/{id}`: Partially update a musician.
- `GET /api/musicians/search?name={name}&profession={profession}`: Search musicians by name or profession.
